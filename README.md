# SPSCAVV - Espaço Viver

Projeto acadêmico desenvolvido com o objetivo de criar um site institucional informativo para auxiliar pessoas no acesso a informações sobre direitos, proteção e canais de denúncia.

## 📌 Sobre o projeto

O site foi desenvolvido para apresentar informações de forma simples e acessível, abordando temas relacionados à proteção social, prevenção de violências e divulgação de canais de apoio.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ⚙️ Funcionalidades

- Interface responsiva
- Alternância entre tema claro e escuro (Dark Mode)
- Organização de conteúdos informativos
- Página dedicada para orientações e canais de denúncia
- Navegação simples e intuitiva

## 🎯 Objetivo

Aplicar conhecimentos de desenvolvimento web adquiridos durante a graduação em Análise e Desenvolvimento de Sistemas, criando uma solução digital com impacto social.
